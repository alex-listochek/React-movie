# React-movie
Веб-приложение для публикации карточек фильмов и их администрирования. 
Коммерческий проект с закрытым исходным кодом.

**Используемый стек технологий:**
- React (Vite + TSX);
- Mantine UI;
- HTML, CSS, JS, TS;
- NodeJS + ExpressJS.
